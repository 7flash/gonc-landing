The Formula

Five dimensions of knowledge

The default reading experience is one-dimensional — you scroll a page top to bottom. We give you five.

1D · Scroll

The Page

You read a book, a file, an article — one direction, top to bottom. This is where everyone starts. One dimension.

2D-3D · Tree

Files & Folders

Files sit in folders, websites link to each other, books sit on shelves. Both your filesystem and the internet are derived from the physical library — books you jump between, shelves you climb up and down. Three dimensions, same since the printing press.

∞ · Canvas

Spatial Freedom

We break through the tree. Every file becomes a card on an infinite canvas. Arrange them the way you think. A minimap gives you context — see everything at a glance, jump to interact.

Space Pan Scroll Zoom G Grid
4th · Layers

Multiple Perspectives

Do you organize fruits by color or by taste? Hierarchies force one perspective. Layers let you see the same files from different angles — each layer is a projection, a different way the pieces relate. This is our key breakthrough.

5th · Time

Navigate History

Git commits are snapshots in time. Navigate them with arrow keys like frames in a film. See how code evolves — inline diffs show what changed.

←→ Navigate commits
∞ · Connections

The Cross-Connections Made Visible

Alt+click a line. Pick a target file. Click a target line. Now there's a permanent, visible link between those two points. This is intertwingularity — the relationships your file tree hides.
