How it works

Four dimensions of code understanding

GitMaps transcends hierarchical file systems with a spatial model that matches how you actually think about code.

1D Lines

Files are one-dimensional

Lines of code, read top to bottom. The atomic unit. Every file is fully rendered as a card you can read, edit, and scroll. This is where everyone starts.

break free
2D Canvas

Canvas breaks hierarchy

Files escape their folders. Arrange them spatially — group related code side by side regardless of directory structure. A minimap gives you overview at a glance. Diff markers on the right edge show what changed.

add depth
3D Layers

Layers add depth

Extract files into focus layers — independently from their folder. Auth, API, UI — each layer is a different plane in 3D space. Switch context without losing position. Connections tie files across layers.

travel through time
4D Time

Git is the fourth dimension

Move through commits while your spatial layout persists. See what changed, where it changed. Inline diffs show additions, deletions, and modifications — review changes spatially instead of one-file-at-a-time.

🗺️

Minimap

Always-on bird's eye view of your entire workspace. Click to navigate. Viewport indicator tracks your position in real-time.

⚡

Canvas Text

Hardware-accelerated text rendering directly on canvas. Thousands of lines rendered at 60fps with LOD transitions — zoomed-out files become compact pills.

📝

CodeMirror Editor

Double-click any card to open the full editor. Syntax highlighting, auto-complete, save, and commit — directly from the canvas.

🔍

Diff Markers

Colored indicators on card edges — green for added, red for deleted, yellow for modified. Navigate between changed hunks with ▲/▼.

🔗

Cross-Layer Connections

Alt+click to draw permanent links between specific lines across files and layers. Connections are the knots that bind your mental model.

📊

Force-Directed Graph

Toggle dependency view (Ctrl+G) to see import relationships as a force-directed layout. Files cluster by coupling.

💻 Local Full power

Edit files, save changes, commit — the full experience.

Requires Bun or Node.js 18+
npx gitmaps

Opens current directory. Use npx gitmaps /path/to/repo for a specific repo.

One-liner — installs Bun if needed
curl -fsSL https://gitmaps.dev/install.sh | bash

Or install manually: bun install -g gitmaps && gitmaps

PowerShell — installs Bun if needed
irm https://gitmaps.dev/install.ps1 | iex

Or install manually: bun install -g gitmaps; gitmaps
