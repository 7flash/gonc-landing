The Canvas

Every file. All at once.

No more Open File → read → close → repeat.

🗺️

Infinite Canvas

Pan, zoom, drag — arrange files the way you think about them. Layout saves automatically.

Space Pan Scroll Zoom G Grid F Expand
📊

Inline Diffs

Green additions, red deletions inside each card. Scrollbar markers show where changes are.

⏱️

Commit Timeline

Arrow keys to navigate commits. See which files changed and how many lines moved.

📌

Persistent Layout

Drag files where you want them. Switch commits, refresh — arrangement stays.

🤖

AI Chat

Press I to open AI sidebar. Ask questions about the commit, get explanations of changes.

⌨️

Keyboard First

Navigate commits, search files, expand cards, arrange and select — no mouse needed.

←→ Commits Ctrl+F Search H Row V Column Ctrl+A Select all

Inline diff with hunk navigation — green markers on the scrollbar show exactly where changes are

Layers

Focus on what matters

Large commits touch dozens of files. Layers let you group the ones that belong together.

Auth Flow 4
Database 6
UI Components 3
+ New Layer
📂

Group related files

Right-click any card → add to a layer. Build focused views of a large commit.

🔄

Switch context instantly

Click a layer tab to show only its files. Each layer remembers its viewport.

🎯

Filtered changed files

Changed Files panel filters to the active layer. Only the files that matter.
