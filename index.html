Features

Everything you need to understand code

Built for developers who think visually. No more tab-switching between files.

🗺️

Infinite Canvas

Pan, zoom, and arrange every file in your repo on a smooth infinite canvas. Drag cards freely, arrange in grids, or let the auto-layout do its thing.

Space Pan Ctrl+Scroll Zoom G Grid
📊

Inline Diffs

See additions in green, deletions in red — right inside each file card. Scrollbar markers show you where changes are at a glance.

🔗

Line-to-Line Connections

Click any line in one file, then click a line in another — instant visual connection. See dependencies as bezier curves across the canvas.

⏱️

Commit Timeline

Navigate commits with arrow keys. See exactly which files changed and how many lines were added or deleted. Each commit is one click away.

📌

Persistent Layout

Drag files where you want them — positions are saved automatically. Switch commits, refresh the page, your layout persists.

⌨️

Keyboard First

Every action has a shortcut. Navigate commits with ←→, search files with Ctrl+F, toggle expand with F, select all with Ctrl+A. Power-user friendly.

←→ Commits Ctrl+F Search F Expand HV Arrange
How It Works

From git log to visual overview

Three steps to see your entire repository at once.

01

Point to your repo

Paste or type any local git repository path. Works with any repo — small or large, any language.

C:\Code\my-project
02

Browse commits visually

Every commit in the timeline shows changed files as file cards on the canvas. Green lines = additions, red = deletions.

03

Connect the dots

Click a line in one file, click a line in another — a connection appears. Map out dependencies, data flow, or just mark related code.
